# 🛒 E-Commerce Website

A modern and responsive **E-Commerce Website** built using **HTML** and **CSS**. This project provides a clean and attractive user interface for an online shopping experience.

---

## 📌 Features

- 🏠 Responsive Home Page
- 🛍️ Product Showcase
- 🔍 Search Bar (UI)
- 📂 Product Categories
- ❤️ Wishlist Icon
- 🛒 Shopping Cart Icon
- 📱 Mobile Responsive Design
- 🎨 Modern User Interface
- 📞 Contact Section
- 📧 Footer with Social Media Links

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/AbhishekChauhan9036/Ecommerce-Website.git
```

### 2. Open the Project

Navigate to the project folder.

### 3. Run the Project

Open the `index.html` file in your preferred web browser.
